# **App Name**: CampusLost

## Core Features:

- Item Submission: Allow users to submit details about found items, including description, location, and contact information.
- Item Search: Enable users to search for lost items using keywords, location, and date.
- Matching Algorithm: Use an AI tool to suggest potential matches between submitted items and search queries based on item descriptions and locations.
- Notification System: Notify users when a potential match for their lost or found item is identified.
- Admin Dashboard: Provide an admin interface to manage items, users, and system settings.

## Style Guidelines:

- Primary color: Muted blue (#6699CC) to evoke a sense of trustworthiness and calm.
- Background color: Very light blue (#E6EEF2), almost white, to provide a clean and uncluttered backdrop.
- Accent color: Soft green (#8FBC8F), used for success messages or found item highlights, as it is analogous to blue but different enough in brightness and saturation.
- Font pairing: 'Inter' (sans-serif) for body text, and 'PT Sans' (sans-serif) for headers to offer a balance of clarity and readability.
- Use simple, clear icons to represent item categories and actions.
- Clean and intuitive layout with a focus on search functionality.
- Subtle animations for loading states and successful item submissions.
