# 🎒 Campus Lost – Smart Lost & Found Management System

Campus Lost is a campus-focused lost and found platform designed to help students and staff quickly report, search, and recover lost items.
By combining intelligent matching, notifications, and an admin management system, Campus Lost reduces friction and improves recovery rates across educational institutions.

🚀 What Campus Lost Does

Campus Lost enables users to:

Submit details of found items

Search for lost items using multiple filters

Receive notifications when potential matches are found

Leverage AI-powered matching to connect lost and found reports

Manage the system efficiently through an admin dashboard

✨ Key Features
📝 Item Submission

Users can submit found items with:

Description

Location

Date

Contact information

Makes item reporting fast and structured

🔍 Item Search

Search lost items using:

Keywords

Location

Date

Designed for quick and accurate discovery

🤖 Matching Algorithm (AI-Powered)

Uses AI to suggest potential matches between:

Lost item searches

Found item submissions

Matches are based on:

Item descriptions

Location proximity

Reduces manual searching effort

🔔 Notification System

Automatically notifies users when:

A potential match is detected

Helps reunite items with owners faster

🛠️ Admin Dashboard

Admins can:

Manage item listings

Manage users

Configure system settings

Ensures platform reliability and moderation
